## DAY-FINDER
This is a web app that give you the day of the week if you give a date(eg. 1/11/2011).
## HOW IT WAS CREATED
- It works on the top of *zeller congurency*.
- The ui is completly done by AI.
## How to run
- open index.html on your browser.
